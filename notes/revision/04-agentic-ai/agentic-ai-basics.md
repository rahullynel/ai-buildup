# Agentic AI Basics

## ReAct loop
- Thought -> Action -> Observation -> repeat.

## Tool use
- Agent can call APIs, search, DB queries, code tools.
- Needs clear stopping criteria and fallback logic.

## Planner -> Worker -> Critic
- Planner decomposes tasks.
- Worker executes tasks.
- Critic validates output quality/gaps.

## Memory types
- Short-term: current session context.
- Long-term: saved preferences/history.
- Vector memory: retrieval over stored notes/docs.

## When agents make sense
- Multi-step workflows.
- Tool-heavy tasks.
- Not needed for simple one-turn Q&A.
