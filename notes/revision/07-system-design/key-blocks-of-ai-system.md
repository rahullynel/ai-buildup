# Key Blocks of an AI System

## Core blocks
- Data ingestion
- Data processing/chunking
- Embedding + indexing
- Retrieval + reranking
- Orchestration/routing
- Prompt construction
- Model inference
- Guardrails/safety
- Evaluation/observability

## High-level flow
User query -> Orchestrator -> Retrieval/API -> Prompt Builder -> LLM -> Answer
